## Hosting, CDN, and edge services

This website uses the services of Cloudflare, Inc. as a Content Delivery Network (CDN), security proxy, and edge computing platform (Cloudflare Workers). As part of these services, Cloudflare technically processes certain connection data (IP address, browser type, pages visited, HTTP headers) to ensure the security, performance, and proper functioning of the website.

Cloudflare is certified under the EU-US Data Privacy Framework and applies the Standard Contractual Clauses for data transfers outside the European Union. For more information on how Cloudflare processes data, please see their privacy policy: https://www.cloudflare.com/privacypolicy/

## Analytics

This website uses Cloudflare Web Analytics to measure traffic. This tool does not set any cookies, does not use fingerprinting, and does not collect data that could personally identify visitors. The data collected (page views, aggregated country of origin, device type) is used solely to understand how the website is used. No further processing of this data is performed.

## Cookies and local storage

This website does not set tracking cookies. Cloudflare may set a strictly necessary technical cookie (`__cf_bm`) for anti-bot protection purposes, which is exempt from consent requirements under CNIL guidelines.

This website uses the browser's local storage (localStorage) to store technical data strictly necessary for the proper functioning of the application, such as your user preferences (e.g. dark or light mode). This data remains stored locally on your device and can be deleted at any time through your browser settings.
